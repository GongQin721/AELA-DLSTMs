# AELA-DLSTMs
AELA-DLSTMs: Attention-Enabled and Location-Aware Double LSTMs for Aspect-level Sentiment Classification

## Requirements
* Python 2.7
* Theano 0.9.0
* numpy 1.13.1
* pygpu 0.6.9
* GloVe.840B.300d

## Running
```
python main_aela.py
```

## Environment
* OS: REHL Server 6.4 (Santiago)
* GPU: NVIDIA GTX 1080
* CUDA: 9.2
* cuDNN: v7.1.4
